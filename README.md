# 🐍 Python Foundations for AI & Machine Learning

Welcome to my personal learning repository! This project serves as a structured sandbox where I document my journey mastering **Python core mechanics**, optimizing data workflows, and implementing foundational **Artificial Intelligence (AI) and Machine Learning (ML)** concepts.

---

## 🧠 Core Python Mechanics
Before diving into models, a deep understanding of Python's underlying architecture is required. This section covers:
* **Object Mutability:** Deep dive into mutable vs. immutable types (e.g., why `str` and `tuple` are immutable, while `list` and `dict` are mutable).
* **Memory Management:** Concepts of variable reassignment vs. in-place mutation, object IDs (`id()`), and string interning.
* **Data Structures:** Efficient use of Lists, Dictionaries, Sets, and Tuples for data manipulation.
* **Comprehensions:** Writing clean, Pythonic code using list, dictionary, and set comprehensions.
* **Functional Programming:** Utilizing `lambda` functions, `map()`, `filter()`, and `reduce()`.

## 🛠 Data Science & ML Stack
This repository transitions foundational Python knowledge into production-ready AI/ML concepts using the standard industry stack:

* **NumPy:** Vectorized operations, N-dimensional arrays, and linear algebra fundamentals.
* **Pandas:** Data ingestion, cleaning, transformation, and DataFrame manipulation.
* **Matplotlib & Seaborn:** Exploratory Data Analysis (EDA) and data visualization.
* **Scikit-Learn:** Implementing basic ML workflows (Preprocessing, Regression, Classification, and Clustering).
